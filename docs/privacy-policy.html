Privacy Policy

Last updated: March 2026

This privacy policy describes how the CalcaLight app (hereinafter "the app") handles user information. The app was developed by Raúl de Pablo with the purpose of turning an iPhone or iPad screen into a light table so that children can trace images onto paper.

1. Data stored by the app

The app stores only basic operational information on the device itself:

  • The number of donations made through the app, in order to personalise the thank-you message.
  • Usage preferences (for example, whether the app is being opened for the first time).

The app does not store any images. Photos are accessed directly from the user's library to be displayed on screen and are discarded when the app is closed.

2. Access to the photo library

The app allows users to select an image from the device's photo library to use as a tracing template. This access is performed through the system photo picker (PhotosUI), which does not grant the app general access to the library: the user explicitly chooses which image to share on each occasion.

The selected image never leaves the device and is never sent to any server.

3. Where data is stored

All data is stored locally on the user's device using the standard system storage (UserDefaults). No proprietary servers or cloud services are used.

4. Use of data

Stored data is used solely for the functioning of the app: displaying the selected image as a light table and personalising the thank-you message after a donation. Data is never used for analytical, advertising, or commercial purposes.

5. Sharing data with third parties

The app does not share any data with third parties. No analytics, advertising, or tracking tools of any kind are integrated.

6. In-app purchases

The app offers optional donations to support development. These transactions are managed entirely by Apple through the App Store. The developer only receives confirmation that a purchase has been completed, without access to any payment data or associated personal information.

7. Deleting data

Users can delete all data stored by the app by uninstalling it from their device. Upon reinstallation, the app will start fresh.

8. Children

CalcaLight is designed to be used by children under adult supervision. The app does not collect personal information from any user, regardless of age. In-app purchases require Apple authentication (Face ID, Touch ID, or passcode), so they remain under the control of the adult responsible for the device.

9. Changes to this policy

This policy may be updated occasionally. The date of the last update appears at the top of this document. Continued use of the app after any change implies acceptance of the new version.

10. Contact

For any privacy-related enquiries, please write to: arenillas_argon.48@icloud.com

本隐私政策描述了 CalcaLight 应用程序（以下简称"本应用"）如何处理用户信息。本应用由 Raúl de Pablo 开发，旨在将 iPhone 或 iPad 屏幕变成描图灯箱，让儿童可以在纸上描绘图像。

本应用仅在设备本地存储基本运行信息：

  • 通过应用完成的捐赠次数，用于个性化感谢信息。
  • 使用偏好设置（例如，是否首次打开应用）。

本应用不存储任何图像。照片直接从用户的相册中读取并显示在屏幕上，关闭应用后即被丢弃。

本应用允许用户从设备相册中选择图像作为描图模板。此访问通过系统照片选择器（PhotosUI）进行，不会向应用授予对相册的一般访问权限：用户每次都明确选择要共享的图像。

所选图像永远不会离开设备，也不会发送到任何服务器。

三、数据存储位置

所有数据均使用标准系统存储（UserDefaults）在用户设备本地存储。不使用任何专有服务器或云服务。

四、数据使用

存储的数据仅用于应用运行：将所选图像显示为描图灯箱，以及在捐赠后个性化感谢信息。数据绝不用于分析、广告或商业目的。

五、与第三方共享数据

本应用不与任何第三方共享数据。未集成任何分析、广告或跟踪工具。

六、应用内购买

本应用提供可选捐赠以支持开发。这些交易完全由 Apple 通过 App Store 管理。开发者仅收到购买完成的确认，无法访问任何支付数据或相关个人信息。

七、删除数据
